# ISICverify - Verify ISIC cards using Node.js

# Install

npm i isicverify

# Usage

const verifyISIC = require('isicverify')
verifyISIC('cardnumber').then(function(isValid) {
	console.log(isValid)
})

# How it works

ISIC does not have a public API, but they have a verification form which this module fills out and then parses the response HTML to get the result
